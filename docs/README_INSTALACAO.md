
## Como instalar no projeto

Copie `GEMINI.md` para a raiz do projeto:

```text
azure-water-quality-data-lake/GEMINI.md
```

Copie a pasta `skills` para a raiz do projeto:

```text
azure-water-quality-data-lake/skills/
```

Estrutura esperada:

```text
azure-water-quality-data-lake
├── GEMINI.md
└── skills
    ├── 01-ingestao-tratamento-qualidade-agua.md
    ├── 02-analise-qualidade-agua.md
    ├── 03-sql-json-dashboard.md
    └── 04-visualizacao-html-awqdl.md
```

## Ordem de uso pelos agentes
